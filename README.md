## 🎯 Professional Online Examination Platform with Advanced Proctoring

### ✨ Features

- 🔐 **Secure Authentication** - Student & Admin login
- 📊 **Excel/CSV Import** - Bulk question upload
- ⏱️ **Custom Timers** - Per-exam time limits
- 🎲 **Randomized Questions** - Different order each attempt
- 🔀 **Shuffled Options** - Prevents answer memorization
- 🛡️ **Anti-Cheat System** - No copy-paste, right-click, tab switching
- 💾 **Auto-Save** - Resume interrupted tests
- 📈 **Analytics Dashboard** - Real-time performance tracking
- 📥 **CSV Export** - Download detailed results
- 📱 **Fully Responsive** - Works on all devices

### 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Storage**: LocalStorage (no database needed)
- **Excel Parsing**: SheetJS
- **Icons**: Font Awesome 6
- **Hosting**: GitHub Pages (free)

### 📊 Data Storage

All data is stored locally in the browser. Each student's results remain on their device unless exported.
